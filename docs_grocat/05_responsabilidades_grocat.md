Responsabilidades identificadas
1. Responsabilidad penal

Podrían existir responsabilidades por:

fraude informático,
acceso ilícito,
lavado de activos,
asociación ilícita.

Actores:

exfuncionarios,
proveedores,
terceros involucrados.
2. Responsabilidad civil

BancoEstado podría enfrentar:

demandas por perjuicios,
compensaciones económicas,
daños reputacionales.
3. Responsabilidad administrativa

La CMF podría:

exigir auditorías,
aplicar sanciones,
solicitar mejoras de control interno.
4. Responsabilidad organizacional

Posibles fallas:

monitoreo insuficiente,
exceso de privilegios,
controles internos débiles,
detección tardía del fraude.
